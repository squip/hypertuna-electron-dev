# indiepress.io — Mini Brand Kit

**Concept**: A clean, modular logo inspired by vintage punk zines: high-contrast, slightly imperfect edges, and a *safety-pin-as-link* motif that doubles as a metaphor for P2P connectivity and censorship resistance.

## Files
- `mark_clean.svg` — stand‑alone logomark, production‑ready
- `mark_rough.svg` — same mark with subtle photocopy roughness (zine vibe)
- `wordmark_strip.svg` — two cut‑paper strips: “indie” and “PRESS·IO”
- `lockup_horizontal.svg` — mark + domain, good for headers
- `favicon.svg` — simplified small‑size mark
- `preview.html` — quick visual overview
- `meta.json` — colors + concept

## Color
- **Ink**: #111111
- **Paper**: #FFFFFF
- **Accent**: #FF2D55

These SVGs use CSS custom properties (`--ink`, `--paper`, `--accent`) so you can theme them easily.

## Typography (swap-in suggestions)
- **Tall compressed (zine headline)**: *Anton*, *Bebas Neue*, *League Gothic*
- **Sans for UI**: *Space Grotesk*, *Inter*, system UI sans

> The SVGs include robust fallbacks and remain usable even if these fonts aren’t loaded.

## Usage
- Use **mark_clean.svg** for app icons, avatars, and stamps.
- Use **mark_rough.svg** on posters, stickers, or when you want a photocopied feel.
- Prefer **lockup_horizontal.svg** for navigation bars and headers.
- The **wordmark_strip.svg** intentionally looks like cut paper; keep some rotation and spacing when you place it.

## Do/Don't
- ✅ Keep it mostly black/white; drop in the accent sparingly.
- ✅ Scale freely — strokes use `vector-effect="non-scaling-stroke"` where relevant.
- ❌ Don’t add drop shadows or gradients; keep the xerox energy.
- ❌ Don’t tightly box the mark; give it air like a sticker on a wall.
